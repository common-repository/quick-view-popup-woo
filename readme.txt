Quick View Popup For WooCommerce brings an array of features to enhance your WooCommerce experience, offering seamless product previews and details.

== Description ==
Elevate your WooCommerce store with a Quick View Popup. Our plugin transforms the customer shopping experience by introducing a quick view feature. Now, shoppers can effortlessly preview product details without navigating away from their current page. Unlock convenience and efficiency for your customers while enhancing your store's usability with Quick View Popup For WooCommerce.

== Key features of Quick View Popup ==

* Easy Access: Display the quick view button on the shop page and all archive pages for easy access.
* Instant Product Access: Quickly view product details by clicking the button.
* Detailed Popup View: Open a popup showcasing specific product details for a comprehensive overview.
* Convenient Action Buttons: Inside the popup, display the "Add to Cart" and "View Details" buttons for seamless shopping. The "Add to Cart" button allows you to add products with quantities directly to the cart, and the "View Details" button navigates you to the single product page for a detailed product view.
* Responsive design: Optimized for various screen sizes, including small devices, ensuring seamless viewing across all resolutions.

== Installation == 
1. First, ensure WooCommerce is installed and activated. If it is not installed, please install [WooCommerce](https://wordpress.org/plugins/woocommerce/). Our plugin will not work without WooCommerce.
2. Download the plugin and then Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
3. Activate the plugin through the 'Plugins' screen in WordPress.
